# UClaw 中文文档

UClaw 是一款跨平台桌面应用，为 OpenClaw AI 智能体提供可视化界面。本文档覆盖安装、配置及所有功能的完整使用说明。

| [安装 UClaw](./installation.md) | 在 macOS、Windows 或 Linux 上下载并安装 |

## 核心功能

| 文档 | 说明 |
|------|------|
| [聊天](./features/chat.md) | 开始对话、@ 指定智能体、使用网络搜索、管理会话 |
| [渠道](./features/channels.md) | 连接微信、Telegram、Discord、飞书、钉钉等消息平台 |
| [定时任务](./features/cron.md) | 创建定时 AI 任务，自动运行并推送结果 |
| [模型](./features/models.md) | 配置 AI 提供商、刷新模型列表、查看 Token 用量 |
| [技能](./features/skills.md) | 启用文档处理、网络搜索等内置技能 |
