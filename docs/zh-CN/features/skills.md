# 在 UClaw 中为 AI 智能体添加和配置技能

> 浏览、启用并配置预置技能，为 UClaw 中的 AI 智能体添加文档处理、网络搜索等能力。

技能是扩展 AI 智能体能力的插件。没有技能时，智能体只能基于其训练数据和你在聊天中输入的内容进行回复。有了技能，智能体可以读取 PDF、处理电子表格、通过专用搜索 API 搜索网络，以及自动发现更多能力。UClaw 附带一套预置技能，首次安装时自动部署并启用——无需单独下载或使用包管理器。

## 预置技能

UClaw 开箱即包含以下技能。它们会在启动时部署到你的技能目录，并默认启用。

### 文档处理技能

| 技能   | 功能                            | 是否需要 API 密钥 |
| ------ | ------------------------------- | ----------------- |
| `pdf`  | 让智能体读取并提取 PDF 文件内容 | 否                |
| `xlsx` | 让智能体处理 Excel 电子表格     | 否                |
| `docx` | 让智能体读取 Word 文档          | 否                |
| `pptx` | 让智能体读取 PowerPoint 演示文稿| 否                |

这四个技能安装后无需额外配置即可立即使用。

### 智能体工具技能

| 技能                   | 功能                                    | 是否需要 API 密钥 |
| ---------------------- | --------------------------------------- | ----------------- |
| `find-skills`          | 帮助智能体在运行时发现并加载更多技能    | 否                |
| `self-improving-agent` | 使智能体能够通过多步骤优化自身回复      | 否                |

### 网络搜索技能

| 技能               | 功能                              | 是否需要 API 密钥              |
| ------------------ | --------------------------------- | ------------------------------ |
| `tavily-search`    | 通过 Tavily Search API 搜索网络   | 是——`TAVILY_API_KEY`           |
| `brave-web-search` | 通过 Brave Search API 搜索网络    | 是——`BRAVE_SEARCH_API_KEY`     |

> **说明：** 网络搜索技能默认启用，但需要 API 密钥才能运行。如果密钥未配置，智能体在尝试使用这些技能时会出现配置错误提示。请在技能页面配置密钥（见下文）。

## 技能目录

UClaw 将托管技能存储在默认目录 `~/.openclaw/skills` 中。技能页面还会从另外两个来源发现技能：

* **工作区技能** — 在你配置的工作区文件夹中找到的技能
* **额外技能目录** — 在高级设置中配置的任何额外技能路径

所有来源统一显示在技能页面上。每个技能条目都显示其实际文件系统位置，以便你在需要检查或修改时直接打开文件夹。

## 启用和禁用技能

技能页面上的每个技能都有一个开关。拨动开关即可启用或禁用技能而不删除它。

**步骤：**

1. 点击左侧导航栏中的**技能**打开技能页面。
2. 浏览列表或按名称搜索。每个条目显示技能名称、启用/禁用状态和来源路径。
3. 点击技能名称旁边的开关进行启用或禁用。更改将在下次对话或定时任务运行时生效。

> **提示：** 禁用你不使用的技能（尤其是未配置密钥的搜索技能）可以防止智能体输出中出现不必要的错误信息。

## 为搜索技能配置 API 密钥

**步骤：**

1. 点击左侧导航栏中的**技能**打开技能页面。
2. 在列表中找到 `tavily-search` 或 `brave-web-search`。
3. 点击技能条目旁边的**配置**或设置图标。
4. 粘贴你的 API 密钥：
   * 对于 `tavily-search`：输入你的 `TAVILY_API_KEY` 值
   * 对于 `brave-web-search`：输入你的 `BRAVE_SEARCH_API_KEY` 值
5. 点击**保存**。密钥将被安全存储并在运行时传递给技能。

> **说明：** 你可以在 [tavily.com](https://tavily.com) 获取 Tavily API 密钥，在 [Brave Search API 门户](https://brave.com/search/api/)获取 Brave Search API 密钥。两者都提供免费套餐。

## 打开技能文件夹

技能页面上的每个技能条目都显示技能文件存储的实际路径。如需检查或排查某个技能：

1. 在技能页面找到该技能。
2. 注意技能名称下方显示的来源路径。
3. 点击**打开文件夹**（如果按钮可用）或在文件管理器中手动导航到该路径。

> **警告：** 直接编辑技能文件可能会破坏其功能。只有在了解相关知识的情况下才应修改技能文件。对于大多数配置需求，请使用技能页面上的配置选项。

## 故障排除

**搜索技能每次运行都返回错误**

最常见的原因是 API 密钥缺失或无效。打开技能页面，点击受影响技能上的**配置**，验证密钥输入是否正确。即使字段看起来已填写，也要重新保存——密钥可能在第一次输入时有误。

**预置技能没有出现在列表中**

UClaw 在启动时将预置技能部署到 `~/.openclaw/skills`。如果技能缺失：
1. 重启 UClaw 触发部署检查。
2. 再次打开技能页面并刷新列表。
3. 如果技能仍未出现，检查**设置 → 网关 → 日志**中启动时是否有错误。

**我禁用了技能但智能体仍在尝试使用它**

技能状态更改对新对话和定时任务运行生效。如果你在活跃对话中，请开启新的聊天会话以使更改生效。
