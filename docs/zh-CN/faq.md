# UClaw 桌面应用常见问题解答

> 关于安装 UClaw、连接 AI 提供商、设置渠道、使用技能和安排任务的常见问题解答。

本页面回答用户最常问到的关于 UClaw 的问题——从它是什么和如何入门，到 AI 提供商的工作方式、可以连接哪些消息渠道，以及如何在 USB 驱动器上携带 UClaw。

---

**UClaw 是什么？**

UClaw 是一款为 OpenClaw AI 智能体提供图形界面的桌面应用程序。你无需在终端中操作或编辑配置文件，而是通过聊天窗口与 AI 智能体交互，通过可视化页面管理渠道和定时任务，并通过设置面板配置一切。

UClaw 在 macOS、Windows 和 Linux 上运行——无需命令行知识。

---

**UClaw 需要 OpenClaw 账号吗？**

不需要。UClaw 不需要单独的 OpenClaw 账号或订阅。你所需要的是至少一个 AI 提供商的 API 密钥——或者你可以使用内置的 New API 端点，UClaw 会通过设置向导中的一键设置为你配置好。

---

**UClaw 支持哪些 AI 提供商？**

UClaw 支持：

* **OpenAI** — 通过 API 密钥或浏览器 OAuth（Codex 订阅）
* **Anthropic** — 通过 API 密钥
* **自定义（OpenAI 兼容）网关** — 任何提供 OpenAI 兼容 API 端点的提供商

所有 API 密钥均存储在你操作系统的原生安全存储中，而非明文文件。

---

**我的 API 密钥安全吗？**

是的。UClaw 将 API 密钥和凭据存储在你操作系统的原生安全存储中：

* **macOS**：Keychain（钥匙串）
* **Windows**：凭据管理器
* **Linux**：Secret Service（例如 GNOME Keyring 或 KWallet）

密钥绝不会以明文形式写入磁盘。

---

**我可以离线使用 UClaw 吗？**

部分功能可以。UClaw 界面本身可以在没有网络连接的情况下启动和浏览。但 AI 回复需要与 AI 提供商 API 的实时连接，网络搜索也需要互联网访问。离线状态下，你仍然可以浏览存储在工作区中的历史对话。

---

**我可以连接哪些消息渠道？**

UClaw 支持以下渠道（在**渠道**页面）：

* 微信（通过内置腾讯插件的二维码流程）
* WhatsApp（二维码流程）
* Telegram（机器人令牌）
* Discord（机器人令牌）
* 飞书 / Lark（应用凭据）
* 钉钉（应用凭据）
* 企业微信（应用凭据）
* QQ 机器人（应用凭据）

每个渠道支持多个账号，每个账号可以绑定到不同的智能体。

---

**如何运行多个智能体？**

每个渠道账号都可以在**渠道**页面绑定到特定智能体。通过该账号到达的消息将由其绑定的智能体处理。

在**聊天**页面，你可以在消息输入框中输入 `@智能体名称` 直接指定某个智能体。UClaw 会切换到该智能体的对话上下文，而非通过默认智能体路由。

---

**为什么点击 X 按钮不会退出 UClaw？**

点击窗口关闭按钮会将 UClaw 隐藏到系统托盘——不会完全退出应用程序。网关和所有已连接的渠道会继续在后台运行。

要完全关闭 UClaw，请点击系统托盘中的 UClaw 图标并选择**退出 UClaw**。

---

**如何更新 UClaw？**

有三种更新方式：

1. **手动检查**：前往**设置 → 关于 → 检查更新**。
2. **自动检查**：在设置中启用自动更新检查，这样 UClaw 会在启动时通知你新版本的可用性。
3. **手动下载**：直接从 [GitHub Releases 页面](https://github.com/DepartureZSH/UClaw/releases)下载最新版本并安装。

注意：在 USB 闪存驱动器上自动更新可能较慢，因为更新涉及大量小文件。USB 3.0+ 驱动器或便携 SSD 可以显著提高更新速度。

---

**我可以将 UClaw 携带在 USB 驱动器上吗？**

可以。UClaw 支持专为 USB 驱动器设计的双分区便携布局。推荐的设置将 macOS `.app` 包保存在 APFS 分区，将 Windows/Linux 版本和共享 UClaw 工作区放在 ExFAT 分区。

使用附带的启动器脚本（而非直接双击应用）以确保无论插入哪台电脑，数据路径都能正确设置。完整的设置指南请参阅[便携 USB 模式](./advanced/portable-usb.md)。

---

**什么是技能？**

技能是扩展 AI 智能体能力的插件。UClaw 预装并默认启用了多个技能，包括文档处理（PDF、Word、Excel、PowerPoint）、网络搜索（Brave 和 Tavily）以及智能体工具技能。

你可以在**技能**页面查看所有已安装的技能、检查其启用状态、在磁盘上打开其文件夹，并输入所需的 API 密钥。某些搜索技能需要自己的 API 密钥（例如 `BRAVE_SEARCH_API_KEY` 或 `TAVILY_API_KEY`）——如果密钥缺失，UClaw 会在调用该技能时显示配置错误。

---

**如何安排 AI 任务？**

前往**定时任务**页面并创建新任务。每个任务包含：

* 用于标识的**名称**
* **触发时间**或间隔
* 智能体将在计划时间运行的**提示词**
* 处理任务的**执行智能体**
* 可选的**发送账号**和**接收目标**，用于将输出推送到消息渠道

对于支持的渠道，UClaw 会自动从渠道目录和会话历史中发现接收目标，因此你无需手动编辑配置文件。
